Skill 简介

自动化流程的最后一步，也是最接近用户的一步。

01

多种桌面 Agent 都能接入

WorkBuddy、阿里悟空、Trae、Claude Code、Codex 等桌面 Agent，可以按各自的 Skill 导入方式调用同一套微信发送能力。

02

扎根 Windows 个人微信桌面版

客户端运行在你的 Windows 11 电脑上，操作已经登录的个人微信桌面版，不需要把联系人和附件交给独立的云端群发平台。

03

接在业务流程的最后一步

上游 Agent 可以整理数据、生成报告、编写通知或完成客户跟进任务，再由这个 Skill 把结果送达指定联系人和群聊。

04

发送前确认，发送后回报

消息、文件、联系人和群聊先预览再执行；完成后回报成功、失败和总数，失败对象可以单独补发。

发送流程

无论使用哪种桌面 Agent，调用逻辑保持一致。

STEP 01 安装激活并登录电脑微信

在 Windows 11 安装客户端，保持个人微信桌面版已登录、电脑未锁屏。

STEP 02 向桌面 Agent 导入 Skill

根据 WorkBuddy、阿里悟空、Trae、Claude Code 或 Codex 的方式导入或配置 Skill。

STEP 03 说清收件人、内容和附件

用自然语言描述联系人、群名、名单、消息和附件，并先预览待执行内容。

STEP 04 确认发送并接收结果

确认后在本机执行，成功、失败和总数清楚回报，失败对象可单独补发。
